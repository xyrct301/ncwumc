## 关于MC
14年底接触到mc，随后便深深喜欢上了这个高度开放且自由的游戏。我对少年时期的回忆就是许多视频博主用他们各自精彩的视频给我带来一段又一段神奇的冒险，让我意识到这个游戏无限的可能性。在22年购买正版，也算是我对这个游戏陪伴我这么多年的一点微薄的回报。

## 关于社团
在贴吧帖子上无意刷到，进到群里后大家都很热情的欢迎，很多老前辈也会在大群里分享他们的经验与生活趣事。这是我第一次感受到mc这款游戏的凝聚力。

## 心愿
希望社团内的大家都能像在mc中的那样，在地球online大型服务器中发挥自己的才能。（自己也要来一场精彩的整合包冒险！ψ(｀∇´)ψ）
