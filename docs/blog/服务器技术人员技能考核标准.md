### 考核说明  

1. **提交要求**  
   - 所有任务需提供`截图/录屏`（包含游戏内效果和配置文件）  
   - 崩溃修复需提交`错误日志+修复过程录屏`  

2. **扣分项**  
   - JAVA版本错误：`-3分`  
   - 模组冲突未解决：`-10分`  

3. **通过标准**  
   - **≥60分**：可独立搭建基础服务器  
   - **≥80分**：具备故障排查和优化能力  
   - **满分**：2小时内完成全部任务  

4. **注意事项** 
   - Jjava -javaagent:**authlib-injector.jar**=https://skin.mualliance.ltd/api/union/yggdrasil -jar server.jar里的**authlib-injector.jar**记得改为具体的版本，如authlib-injector-1.2.3.jar
